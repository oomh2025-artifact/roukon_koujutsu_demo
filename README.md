# 労働衛生コンサルタント 口述試験対策システム

労働衛生コンサルタント（保健衛生）口述試験の対策用チャットボットです。

## 🎮 使い方

1. `index.html` をブラウザで開く（またはGitHub Pagesでホスティング）
2. モードを選択して学習開始

## 📚 モード説明

| モード | 説明 |
|--------|------|
| **本編** | 基本→詳細→応用の3段階構成。詳細解説付き |
| **追加問題編** | 補足的な知識を補強 |
| **復習フォルダ** | 5点以下の問題を自動保存。繰り返し復習 |

## 📋 試験ルール

- 5つのテーマについてランダムに出題
- 各問題は10点満点でAI採点
- 6点以上で次のレベルへ進行
- 5点以下は補足質問＋復習フォルダに自動追加
- 「持ち帰り」は2回まで使用可能
- 合格基準：60%以上

## 🌐 GitHub Pagesでの公開方法

1. GitHubでリポジトリを作成
2. このフォルダの内容をプッシュ
3. Settings → Pages → Source: Deploy from a branch → main / root
4. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]/` で公開

## 📁 ファイル構成

```
/
├── index.html      # アプリ本体
├── data/
│   ├── questions.md   # 本編問題
│   └── followup.md    # 追加問題
└── README.md       # このファイル
```

## 📝 問題の更新方法

問題を更新する場合は、`data/` フォルダ内のマークダウンファイルを編集してください。

### 本編問題（questions.md）の形式

```markdown
## テーマ名

**Q1（基本問題）：**
問題文

**【模範解答】**
解答文

**【この問題で試験官が見ているポイント】**
- ポイント1
- ポイント2

---

**Q2（詳細問題）：**
...
```

### 追加問題（followup.md）の形式

```markdown
### Q：質問文

**A：** 回答文

---
```

## ⚠️ 注意事項

- インターネット接続が必要です（AI採点にAPIを使用）
- 復習フォルダはブラウザのlocalStorageに保存されます
- ブラウザのデータを削除すると復習フォルダの内容も消えます
